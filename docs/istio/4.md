# 应用篇

## virtual service

- 在每个服务的chart中为每个服务部署一个vs，使得其他服务能够调用自己；如果只是简单的先支持被调用，不需要有subset；subset需要引用destination rule
- 设置超时时间

## destination rule

## service entry

被注入的服务中需要访问外部服务，在不配置service entry时，代码中使用http.Get拿到的返回是nil。需要在service entry中配置该外部服务，服务才能正常和外部通信获取正常的response

## envoy filter

对后端微服务进行限流
